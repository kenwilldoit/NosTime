### 本サービスの概要（700文字以内）
本サービスは、ユーザーが自分の世代ごとに分類された興味のあるジャンルを選択し、過去に流行したモノを久しぶりに思い出してもらうことで、脳内にドーパミンを分泌させ、その瞬間だけでも脳にポジティブな刺激を与えることを目的としています。
MVP段階では、サイト内に見当たらない製品があった場合、ユーザー自身が新しく投稿できる仕組みを提供しています。また、他ユーザーの投稿に対して、ブックマーク、いいね、コメントといった基本的なリアクション機能を備えており、懐かしさをきっかけとした共感や交流が生まれる場を目指しています。
